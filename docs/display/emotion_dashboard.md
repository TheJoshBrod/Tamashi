# Emotion Dashboard

`localhost:8000/display/`

Displays an animated avatar face whose state is driven by the `EmotionManager` via WebSocket. The avatar mimics a lifelike presence by dropping into `GROGGY` and `ASLEEP` states at night, and populates whimsical thought bubbles if a `MEMORY_DREAM` fires during sleep. 

For details on how emotions map to UI states and how to add new ones, see the [Extension Guide](../extending_tamashi.md) and [Implementation Details](../implementation.md).
