# Display & UI Directory

This directory contains documentation for the two main visual interfaces of Tamashi, as well as the test scripts for generating graphical topologies.

- [Emotion Dashboard](emotion_dashboard.md) — The real-time, animated avatar interface responsive to the agent's internal state.
- [Memory Cartography](memory_cartography.md) — The full-screen interactive node graph used to visualize and edit long-term memory.
- [Testing Scripts](testing_scripts.md) — Helper utilities for populating the graph with stress-test topologies.
